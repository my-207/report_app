🔧 核心服务模块 & 调用关系

docxService
→ python-docx unpack
→ python-docx pack
→ PowerShell fallback
← 被调: fillerService
statementsParser
→ JSON-LD 解析
→ 实体识别
→ 属性映射 (rj-property-map)
← 被调: fillerService
templateService
→ 模板上传/缓存
→ 占位符分析
→ 表头提取
← 被调: fillerService, apiRoutes
chapterExtractor
→ MD.docx 解析
→ 章节分组
→ 键值对/签名提取
← 被调: fillerService
fillerService
→ fillBySubtreeCopy
→ replacePlaceholders
→ fillTableFromSource
← 调度所有填充子模块
subtreeInserter
→ 表头匹配
→ XML 子树复制
→ 行模板克隆
← 被调: fillerService
dataService
→ JSON/YAML 解析
→ 数据校验
→ 预览生成
← 被调: apiRoutes
taskService
→ 状态机调度
→ 异步执行
→ Webhook 回调
← 被调: employeeRoutes

数据提取路径

📥 输入 → 知识层

分子公司报告→文档解析→本体建模

.rj + MD.docx→双源合并→统一数据模型
